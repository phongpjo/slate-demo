    Introduction

    Following module

    Follow

    This is example curl:

    curl --location --request POST 'https://apiextsit.tcbs.com.vn/icopy/relation/v1/copiers/follow' </span>
--header 'Authorization: Bearer JWT' </span>
--header 'Content-Type: application/json' </span>
--data-raw '{
 "followerTcbsId": "0001409989",
 "traderId": 123
}'

    This is example response:

    {
  "followerId": 0,
  "traderId": 0
}

    HTTP Request

    POST /account-relation/e/v1/copiers/follow

    Parameters

    Name Description
    reqDto *
    object
    (body)    		 request dto

    Response

    Trường Cách tính Mô tả
    followerId ACCOUNT_RELATION.ID follow ID
    traderId TRADER_ACCOUNT.ID trader ID

    Rating module

    Hiển thị thông tin đánh giá trader

    This is example curl:

    curl --location --request GET 'https://apiextsit.tcbs.com.vn/icopy/relation/v1/traders/0001098985/rating?ratingTcbsId=0001409989' </span>
--header 'Accept-language: vi' </span>
--header 'Content-Type: application/json' </span>
--header 'Authorization: Bearer JWT'

    This is example response:

    {
  "avgScore": 0,
  "code": "VERY_GOOD",
  "data": {
    "content": [
      {
        "avatar": "string",
        "critical": "You are so dep!",
        "fullname": "D T L",
        "ratingId": 0,
        "ratingRole": "[COPIER, SUBSCRIBER]",
        "ratingTcbsId": "0001000012",
        "replies": [],
        "score": 3,
        "updatedTime": "yyyy-MM-dd HH:mm:ss"
      }
    ],
    "first": true,
    "hasNext": true,
    "hasPrevious": true,
    "last": true,
    "number": 0,
    "numberOfElements": 0,
    "size": 0,
    "totalElements": 0,
    "totalPages": 0
  },
  "rankCode": "string",
  "tcbsId": "string"
}

    HTTP Request

    GET /account-relation/e/v1/traders/{tcbsID}/rating

    Parameters

    Name Description
    tcbsID *
    string
    (path)    		 tcbsID
    ratingTcbsId
    string
    (query)    		 ratingTcbsId
    ratingRole
    string
    (query)    		 ratingRole
    page
    integer($int32)
    (query)    		 page
    Default value : 0
    size
    integer($int32)
    (query)    		 size
    Default value : 0

    Response

    Trường Cách tính Mô tả
    avgScore icopy.customerRatingSummary.totalScore / icopy.customerRatingSummary.totalScoring điểm trung bình
    code code đánh giá dựa theo điểm TB
    rankCode null
    tcbsId CUSTOMER_RATING.TCBSID TCBS ID của trader
    data.first là trang đầu tiên phân trang
    data.hasNext có trang tiếp theo phân trang
    data.hasPrevious có trang liền trước phân trang
    data.last là trang cuối phân trang
    data.number số trang phân trang
    data.numberOfElements số lượng trong content phân trang
    data.size kích thước trang phân trang
    data.totalElements tổng số phần tử phân trang
    data.totalPages tổng số trang phân trang
    data.content.avatar icopy.traderListingItem.accountInfo.avatar avatar người đánh giá
    data.content.critical CUSTOMER_RATING.CRITICAL nội dung đánh giá
    data.content.fullname icopy.traderListingItem.accountInfo.fullname tên người đánh giá
    data.content.ratingId CUSTOMER_RATING.ID ID
    data.content.ratingRole CUSTOMER_RATING.RATING_ROLE role (COPIER/SUBSCRIBER )
    data.content.ratingTcbsId CUSTOMER_RATING.RATING_TCBSID TCBSID của trader
    data.content.score CUSTOMER_RATING.SCORE điểm
    data.content.updatedTime CUSTOMER_RATING.UPDATED_TIME timestamp
    data.content.replies MESSAGE_REPLY danh sách các reply của trader, object tương tự Phản hồi đánh giá của trader
